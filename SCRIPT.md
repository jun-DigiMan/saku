## ❓ Q&A 切り返し

**「Calendlyと何が違うの？」**
Calendlyは顧客が自分で選ぶ外向けツールです。サクピタは複数メンバーのカレンダーを見ながら担当者が能動的に調整する社内向けで、メール・スプシ・リマインダーまで一体化しています。

**「ダブルブッキングは起きない？」**
理論上は1〜2秒の競合リスクはゼロではないですが、営業チームで使うツールなので同時操作のリスクは極めて低く、現状は許容設計にしています。

**「コストが本当にゼロ？」**
GitHub Pagesは無料、Google APIは無料枠内です。営業チームの商談件数では枠を超えません。

**「バニラJSで5500行は保守が大変では？」**
今回はサーバーレス・依存ゼロを最優先にした設計判断です。フレームワーク依存ゼロなので5年後も確実に動きます。
